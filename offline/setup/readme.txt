---------------------------------------

System Requirements

1. 2GB of hard drive space
2. 4GB RAM (6GB recommended)
3. Intel pentium 4 and above
4. Internet for first time setup


Install Instructions

1. Run setup.exe to start installation.
2. Click on Create desktop icon checkbox.
3. Wait for installation to complete.
4. After installation run message from desktop or open message.bat from installation.

If you have any problems reach out to distrbutor. Online support will be enabled soon!
